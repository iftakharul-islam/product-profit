== Installation ==

1. Upload the `product-profit` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to WooCommerce > Profit/Loss Report to access the reports.

== Configuration ==

### Setting Buy Prices
1. Edit any product in WooCommerce
2. Look for the "Buy Price" field in the product data section
3. Enter the cost price for the product
4. Save the product

### Generating Reports
1. Navigate to **WooCommerce > Profit/Loss Report**
2. Select your desired date range
3. Choose category filter (optional)
4. Select sorting options
5. Click "Generate Report"

### Export Options
- **Export Summary CSV**: Download overall totals
- **Export Detailed CSV**: Download per-product breakdown

### Email Reports
The plugin automatically sends weekly profit summary emails to the admin email address. You can modify the frequency in the plugin code.

== Frequently Asked Questions ==

= How do I set buy prices for my products? =

Edit any product in WooCommerce and you'll see a "Buy Price" field in the product data section. Enter the cost price for that product and save. This buy price will be used to calculate profits.

= Can I filter reports by product categories? =

Yes! In the Profit/Loss Report page, you can select specific product categories from the dropdown menu to filter your reports.

= What date ranges can I use for reports? =

You can select any custom date range using the start and end date pickers. The plugin will calculate profits for all completed orders within that period.

= How do I export my reports? =

Use the "Export Summary CSV" button for overall totals or "Export Detailed CSV" for per-product breakdown. The CSV files will download automatically.

= Are the reports real-time? =

Yes, the reports are generated in real-time based on your WooCommerce orders and product buy prices.

= Can I sort the product list? =

Yes, you can sort by product name, total sales, total buy price, profit, or quantity sold in ascending or descending order.

= How often are email reports sent? =

By default, profit summary emails are sent weekly. You can modify this in the plugin code to daily, weekly, or monthly.
